# UVindex-bot

#### A Telegram bot to check UV index by using the location you sending to the bot.

#### Technologies:
 - Node.js;
 - Telegram Bot API;
 - Telegraf library;
 - Weather API.
 
 
 #### How to use:
 - Open [UVIndex-bot](https://t.me/UVindex_bot);
 - Click 
/start
;
 - Send your location to the bot via attachments.
